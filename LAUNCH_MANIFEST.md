## Emergency Procedures

### Common Issues & Solutions

**Issue**: Pi SDK Not Loading
- **Cause**: HTTP instead of HTTPS
- **Solution**: Ensure GitHub Pages HTTPS is enabled

**Issue**: Wallet Connection Fails
- **Cause**: Not using Pi Browser
- **Solution**: Instruct users to use official Pi Browser app

**Issue**: Payment Errors
- **Cause**: Insufficient balance or network issues
- **Solution**: Verify wallet funds and network connectivity

**Issue**: Resources Not Loading
- **Cause**: Incorrect file paths or 404 errors
- **Solution**: Verify all file references and paths

### Rollback Plan
If critical issues arise:
1. Identify the issue via GitHub Issues
2. Fix in a new branch
3. Test thoroughly
4. Deploy update via git push
5. Monitor for 24 hours
